wujSDL

A Pygame wrapper with boost and easy API

Why wujSDL?

  • Fast rendering
  • Easy syntax
  • Can enable ModernGL easily
  • Open source

API

import wujsdl as sdl square = sdl.square(32, 32, "#FF0000", ((800-32) // 2, (600-32) // 2)) sdl.go("800x600 #FFFFFF Wujlav's Pygame Window")
